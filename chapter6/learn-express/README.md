# Express 프레임워크

Node.js로 백엔드를 구성할 때 필요한 프레임워크 중에서 가장 독보적인 라이브러리이다.

**bin/www.js**
> 서버구동 (콜백: app.js)
> 에러리스너(콜백: 콘솔에 로그 출력)

**app.js**
> express 프레임워크 객체 생성 (변수명: `app`)
> 프레임워크 뷰엔진 설정 (`app.set`)
> 프레임워크 미들웨어 장착(app.use)

**routes/**
> 라우팅 담당
> **view/**에서 REST API 기반 메서드로 요청이 들어올 때 만나게 될 경로들

**view/**
> 템플릿 엔진(pug)로 작성됐고 클라이언트가 만나게 될 화면
